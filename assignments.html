UAS mapping will be done at the Lake Wheeler COA: Areas with NGAT permit for sUAS mapping

Data

  • North Carolina data set used in MEA582
  • boundaries of the Lake Wheeler (LW) COA
  • sUAS surveys of LW area: images, GCPs, log files, DSMs

  • 2001 lidar data (NC Floodplain mapping - for SECREF it is in the MEA582 data set)

Software

  • GRASS GIS 7.0.1 (you can use these instructions to install it)
  • Trimble flight planning software (provided at NCSU)
  • Agisoft Photoscan (provided at NCSU)
  • OpenDroneMap (installation will be discussed in class)

How to install Trimble Aerial Imaging software

  1. Download the installation file.
  2. The installer will guide you to install Trimble Installation Manager on your computer.
  3. Open the Trimble Installation Manager.
  4. Click on the Controller Icon (on the far left pane).
  5. Click on "Show All" , at the bottom, right of Help..
  6. Choose the Aerial Imaging and click Install (right bottom corner).
  7. Click on the Aerial Imaging icon and wait for it to contact the server. Eventually the center-right pane populates with three tabs. I selected the Try Software tab. The bottom right button label now reads "Create license".
  8. In the next step you need to provide the username and password for the Trimble account, if you don't have one, you need to create it and then log in.
  9. Download should start automatically after logging in.
